# Using my creations

If you wish to use code/sprites/music, these are the following terms:

### Code

Free to use! You can use my code to learn how mindustry mods are made, or copy it for your mods.

(Free to use unless specified otherwise inside the file)

### Sprites

<ins>You cannot copy my sprites. You *can not* add them into your projects with minor modifications. </ins>

You can use my sprites as a learning reference, but NEVER I repeat NEVER EVER directly copy them.

### Music

The title screen music is made by Haznobrain. Ask them if you can.

The boss music that (may or may not be) in the mod currently is made by Megacheb, ask them if you can use it.

### Audio

The audio I find is usually from royalty free sites such as freesound or pixabay. Some are made by me. The best course of action is to just ask me.
